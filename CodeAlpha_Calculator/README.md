Vector Core is a high-performance "Cyber-OS" engineering console. It features a professional Glassmorphism UI with a Ruby Red and Cyan aesthetic, optimized for both daily arithmetic and advanced scientific logic.

✨ Features
🔴 Vector UI: Dark-mode "Command Bar" layout with glowing hover effects.

📜 Logic Log: Sliding sidebar to track and clear past calculation history.

🧪 Advanced Suite: Integrated support for sin, cos, log, x², and √.

⌨️ Hardware Link: Full support for physical Numpad and Enter key inputs.

⚡ Smart Engine: Real-time equation parsing with auto-closing parentheses.

🛠️ Tech Stack
HTML (Semantic Grid & Structure) 🏗️

CSS (Glassmorphism & Neon Effects) 🎨

JavaScript (ES6 Mathematical Parser) 🧠
